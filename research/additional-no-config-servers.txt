# Additional No-Config MCP Servers

## CALCULATOR/MATH SERVERS (No API Required)

### 1. wrtnlabs/calculator-mcp
- Available via: `npx @wrtnlabs/calculator-mcp`
- Features: Basic arithmetic operations
- No configuration needed

### 2. simonberner/mcp-server-calculator  
- Basic math operations demo server
- Simple add/subtract/multiply/divide
- Local development friendly

### 3. varunidealabs/simple-calculator-mcp-server
- Fast arithmetic calculation tools
- Four basic operations (add, subtract, multiply, divide)
- Designed for AI assistants
