# MCP Servers with No Required Configuration

## Confirmed No-Config Servers (Research Verified)

### Official Reference Servers
1. **everything** - `@modelcontextprotocol/server-everything`
   - No environment variables required
   - Command: `npx -y @modelcontextprotocol/server-everything`
   - Description: Reference/test server with prompts, resources, and tools

2. **fetch** - `@modelcontextprotocol/server-fetch`
   - No required environment variables or API keys
   - Optional flags: --ignore-robots-txt, --user-agent, --proxy-url
   - Command: `uvx mcp-server-fetch` or `pip install mcp-server-fetch`
   - Description: Web content fetching and conversion

3. **time** - `@modelcontextprotocol/server-time`
   - No required environment variables
   - Optional: --local-timezone flag
   - Command: `uvx mcp-server-time`
   - Description: Time and timezone conversion capabilities

4. **memory** - `@modelcontextprotocol/server-memory`
   - Likely no config required (need to verify)
   - Description: Knowledge graph-based persistent memory system
